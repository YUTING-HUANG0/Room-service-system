# 🎥 房務自動化系統 - 演示腳本 (Demo Script)

## 1. 簡介 (Intro)
- **畫面**: 首頁 (Landing Page)
- **旁白**: 「大家好，這是一個專為旅宿業者打造的『房務自動化中控平台』。它解決了傳統民宿最頭痛的兩件事：重複接單與房務溝通效率低落。」

## 2. 核心功能展示 - iCal 同步 (Sync Engine)
- **畫面**: 後台 `/admin/rooms` -> 點擊房號
- **操作**: 展示貼上 Agoda/Booking 連結的欄位。
- **旁白**: 「首先是核心同步引擎。系統透過 iCal 協定，每 15 分鐘自動將 Agoda 與 Booking.com 的訂單同步進來，並自動阻擋重複預訂。」

## 3. 客人訂房 (Guest Booking)
- **畫面**: 前台 `/booking`
- **操作**: 
    1. 選擇日期 (例如明天)。
    2. 顯示空房 -> 預訂。
    3. 填寫姓名 "王小明" -> 送出。
- **畫面**: 手機 LINE 跳出通知 (合成或實拍)
- **旁白**: 「當客人在官網下單，老闆的手機立刻收到 LINE 通知，完全不錯過任何商機。」

## 4. 房務自動化流程 (Housekeeping Workflow)
- **畫面**: 老闆後台 `/admin/tasks`
- **旁白**: 「接著是房務流程。當系統檢測到退房時，會自動生成清潔任務。」
- **畫面**: 切換到手機版面 (模擬房務員 App) `/housekeeper`
- **操作**: 
    1. 房務員看到待接任務 -> 點擊「搶單」。
    2. 點擊「上傳照片」 -> 選擇一張乾淨房間照。
    3. 點擊「完工」。
- **旁白**: 「房務員用手機就能看到今天該掃哪間，掃完直接拍照回傳。」

## 5. 老闆驗收 (Admin Review)
- **畫面**: 切回後台 `/admin`
- **操作**: 點擊「驗收」按鈕。
- **旁白**: 「老闆在後台檢查照片，確沒問題後點擊驗收，房間狀態就會自動變更為『乾淨 (Clean)』，隨時準備迎接下一組客人。」

## 6. 結尾 (Outro)
- **畫面**: 儀表板 (Dashboard)
- **旁白**: 「這套系統透過自動化與即時通知，大幅降低了 50% 的溝通成本，讓管理變得輕鬆簡單。謝謝大家。」
